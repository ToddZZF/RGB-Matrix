# RGB-Matrix

DIY a RGB matrix from scratch

从0自制一个RGB矩阵灯（像素灯）

> 注：并不是完全“从0”，部分难以DIY的东西从淘宝上获取。

## 功能

- [ ] 基础功能
  - [ ] 能亮（就算成功）
- [ ] 扩展功能
  - [ ] 电子表
    - [ ] 时、分、秒
    - [ ] 星期几
    - [ ] 年、月、日
    - [ ] 计时
    - [ ] 倒计时
    - [ ] 闹钟
  - [ ] 传感器
    - [ ] 温湿度
      - [ ] 显示数值
    - [ ] 亮度
      - [ ] 调节背光
      - [ ] 显示数值
    - [ ] 麦克风
      - [ ] 频谱仪
      - [ ] 语言控制
  - [ ] 信息板
    - [ ] IP
    - [ ] WiFi信号强度
    - [ ] 天气
    - [ ] B站/微博/知乎粉丝数
    - [ ] 喝水提醒/久坐提醒
    - [ ] 番茄钟
  - [ ] 其他输入
    - [ ] 按键
    - [ ] 旋钮
    - [ ] 键盘/鼠标
    - [ ] TF卡
  - [ ] 其他输出
    - [ ] 扬声器
    - [ ] 蓝牙小键盘
    - [ ] 红外遥控
  - [ ] 其他
    - [ ] 外面采用原子镜可以当镜子用
    - [ ] 磁吸组合成大的矩阵灯
    - [ ] 与电脑连接当氛围灯
